This skill has moved to the [agentic-plugins-marketplace](https://github.com/hmcts/agentic-plugins-marketplace) as part of the `openspec` plugin (bundled with `openspec-explore`, `openspec-propose`, and `openspec-apply-change`).

Install with:

```
/plugin install openspec@agentic-plugins-marketplace
```

**Prerequisite**: the `openspec` CLI must be on your `PATH`.

The `/opsx:archive` slash command in this repo still works and contains the same logic inline — the skill and the command are parallel implementations.
